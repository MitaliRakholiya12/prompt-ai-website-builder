# 🤖 AI Website Builder — Frontend UI

A modern, responsive frontend for an AI-powered website builder platform. Users can describe their project idea in natural language and get a fully generated website — built with React, Vite, and Framer Motion.

## ✨ Features

- 🧠 **Hero Section** — Prompt input box where users describe their project idea in natural language
- ⚙️ **How It Works** — 3-step flow: Chat → AI Understanding → Customization
- 🌐 **Multi-Language Support** — Tech icons (React, Node.js, Python, Flutter, Docker, MySQL) with horizontal & vertical scroll interactions
- 💳 **Pricing Section** — Starter, Pro & Enterprise plan cards with Monthly/Yearly toggle
- ❓ **FAQ Section** — Expandable accordion questions for user guidance
- 🎞️ **Smooth Animations** — Framer Motion transitions throughout
- 📱 **Fully Responsive** — Works seamlessly on all screen sizes

## 🚀 Getting Started

```bash
# 1. Clone the repository
git clone https://github.com/MitaliRakholiya12/prompt-ai-website-builder.git

# 2. Install dependencies
npm install

# 3. Start development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
